# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/),
проект придерживается [Semantic Versioning](https://semver.org/).

## [0.1.2] - 2026-04-13

### Fixed

- Исправлен запуск контейнера: `pyproject.toml` и `RELEASE_NOTES.md` теперь копируются в образ, иначе `get_version()` падал с `FileNotFoundError` на старте.

## [0.1.1] - 2026-04-13

### Internal

- Skill `/release` теперь обязательно прогоняет ruff, mypy и pytest перед публикацией релиза.
- Мелкая чистка форматирования в сервисе рассылки релизов.

## [0.1.0] - 2026-04-12

### Added

- Редактирование настроек чата прямо в админ-карточке: активация бота, тихие часы, фиксированное звание рулетки, автоматический ежедневный розыгрыш.
- Инфраструктура релизов: версия проекта в `pyproject.toml`, `CHANGELOG.md`, `RELEASE_NOTES.md`, автоматическая рассылка обновлений при старте бота, skill `/release`.

### Changed

- README сокращён и переработан.

### Internal

- Удалён неработающий фильтр цензуры ответов LLM и все связанные настройки.
- Правки CI: статический анализ (ruff, mypy), синхронизация версии Python, стабилизация CD-логики после прогона тестов.
